# Simple Q&A Chatbot (MSCS-633 Hands-On 3)

This project contains a terminal chatbot built with Python and ChatterBot, with Django installed as part of the assignment environment requirements.

## Files
- `chatbot.py`: Main chatbot application.
- `requirements.txt`: Python dependencies.
- `manifest.txt`: Deliverable manifest.
- `run_chatbot.bat`: Windows launcher script.

## Setup
- Run `run_chatbot.bat`; it auto-creates `.venv` and installs dependencies when needed.
